### –í –∫–∞–∫–æ–º —Ñ–∞–π–ª–µ –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ —Å–º–æ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∫–∞—Ç–∞–ª–æ–≥–∞—Ö –≤ —Ñ–∞–π–ª–æ–≤—É—é —Å–∏—Å—Ç–µ–º—É?

### –ß—Ç–æ —Ç–∞–∫–æ–µ inode? –ö–∞–∫–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è —Ç–∞–º —Ö—Ä–∞–Ω–∏—Ç—Å—è?

Inode (–∏–Ω–¥–µ–∫—Å–Ω—ã–π –¥–µ—Å–∫—Ä–∏–ø—Ç–æ—Ä) - —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –≤ –∫–æ—Ç–æ—Ä–æ–π —Ö—Ä–∞–Ω—è—Ç—Å—è –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–µ —Ñ–∞–π–ª–∞ –∏ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã –±–ª–æ–∫–∏ —Å –¥–∞–Ω–Ω—ã–º–∏ —Ñ–∞–π–ª–∞. –•—Ä–∞–Ω–∏—Ç –≤—Å—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é, –∫—Ä–æ–º–µ –∏–º–µ–Ω–∏ —Ñ–∞–π–ª–∞ –∏ –¥–∞–Ω–Ω—ã—Ö. –ö–∞–∂–¥—ã–π —Ñ–∞–π–ª –≤ –¥–∞–Ω–Ω–æ–º –∫–∞—Ç–∞–ª–æ–≥–µ —è–≤–ª—è–µ—Ç—Å—è –∑–∞–ø–∏—Å—å—é —Å –∏–º–µ–Ω–µ–º —Ñ–∞–π–ª–∞ –∏ –Ω–æ–º–µ—Ä–æ–º –∏–Ω–¥–µ–∫—Å–∞. –í—Å—è –æ—Å—Ç–∞–ª—å–Ω–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ —Ñ–∞–π–ª–µ –∏–∑–≤–ª–µ–∫–∞–µ—Ç—Å—è –∏–∑ —Ç–∞–±–ª–∏—Ü—ã –∏–Ω–¥–µ–∫—Å–æ–≤ –ø—É—Ç–µ–º —Å—Å—ã–ª–∫–∏ –Ω–∞ –Ω–æ–º–µ—Ä –∏–Ω–¥–µ–∫—Å–∞. –ù–æ–º–µ—Ä–∞ inodes —É–Ω–∏–∫–∞–ª—å–Ω—ã –Ω–∞ —É—Ä–æ–≤–Ω–µ —Ä–∞–∑–¥–µ–ª–∞. –ö–∞–∂–¥—ã–π —Ä–∞–∑–¥–µ–ª –∫–∞–∫ —Å–æ–±—Å—Ç–≤–µ–Ω–Ω–∞—è —Ç–∞–±–ª–∏—Ü–∞ –∏–Ω–¥–µ–∫—Å–æ–≤. –ï—Å–ª–∏ —É –≤–∞—Å –∑–∞–∫–æ–Ω—á–∏–ª–∏—Å—å inode, –≤—ã –Ω–µ –º–æ–∂–µ—Ç–µ —Å–æ–∑–¥–∞–≤–∞—Ç—å –Ω–æ–≤—ã–µ —Ñ–∞–π–ª—ã, –¥–∞–∂–µ –µ—Å–ª–∏ —É –≤–∞—Å –µ—Å—Ç—å —Å–≤–æ–±–æ–¥–Ω–æ–µ –º–µ—Å—Ç–æ –Ω–∞ –¥–∞–Ω–Ω–æ–º —Ä–∞–∑–¥–µ–ª–µ.

Inodes —Ö—Ä–∞–Ω–∏—Ç –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–µ –æ —Ñ–∞–π–ª–µ, –∫ –∫–æ—Ç–æ—Ä–æ–º—É –æ–Ω –æ—Ç–Ω–æ—Å–∏—Ç—Å—è. –≠—Ç–∏ –º–µ—Ç–∞–¥–∞–Ω–Ω—ã–µ —Å–æ–¥–µ—Ä–∂–∞—Ç –≤—Å—é –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ–± —É–∫–∞–∑–∞–Ω–Ω–æ–º —Ñ–∞–π–ª–µ.

- –†–∞–∑–º–µ—Ä.
- –†–∞–∑—Ä–µ—à–µ–Ω–∏–µ.
- –í–ª–∞–¥–µ–ª–µ—Ü/–≥—Ä—É–ø–ø–∞.
- –†–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏–µ –∂–µ—Å—Ç–∫–æ–≥–æ –¥–∏—Å–∫–∞.
- –î–∞—Ç–∞/–≤—Ä–µ–º—è.
- –õ—é–±–∞—è –¥—Ä—É–≥–∞—è –Ω–µ–æ–±—Ö–æ–¥–∏–º–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è.

### –ß—Ç–æ —Ç–∞–∫–æ–µ hard link? –í —á–µ–º —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É hard link –∏ soft link? –ü—Ä–∏–º–µ—Ä—ã.

**Hard link**:
–°—Å—ã–ª–∫–∞ –Ω–∞ —Ñ–∞–π–ª –≤ —Ñ–∞–π–ª–æ–≤–æ–π —Å–∏—Å—Ç–µ–º–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Ç–∞–∫–æ–≥–æ –∂–µ inode –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞, –∫–∞–∫ —É —Ñ–∞–π–ª–∞, –Ω–∞ –∫–æ—Ç–æ—Ä—ã–π —Å—Å—ã–ª–∞–µ–º—Å—è.
–°–æ–∑–¥–∞–¥–∏–º —Ñ–∞–π–ª _realFile_.

```

```

–°–æ–∑–¥–∞–¥–∏–º hard link –∫–æ–º–∞–Ω–¥–æ–π `ln <—Ü–µ–ª–µ–≤–æ–π_—Ñ–∞–π–ª> <—Ñ–∞–π–ª_—Å—Å—ã–ª–∫–∞>`:

–ü—Ä–æ–≤–µ—Ä–∏–º, —á—Ç–æ inode —É —Ñ–∞–π–ª–∞ _realFile_ –∏ hard —Å—Å—ã–ª–∫–µ _hardLink_ –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä.

–∏—Ç–æ–≥–æ 0

```

–ö–∞–∫ –≤–∏–¥–Ω–æ realFile –∏ hardLink –∏–º–µ—é—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä inode.

**Soft link**:
–°–æ–∑–¥–∞–¥–∏–º soft —Å—Å—ã–ª–∫—É –Ω–∞ —Ñ–∞–π–ª _realFile_.

–ü—Ä–æ–≤–µ—Ä–∏–º, —á—Ç–æ —á–∏—Å–ª–æ–≤–æ–π –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä _softLink_ –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –æ—Ç —á–∏—Å–ª–æ–≤–æ–≥–æ –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞ _realFile_.

```

–ù–µ–∫–æ—Ç–æ—Ä—ã–µ –Ω—é–∞–Ω—Å—ã:
- Soft —Å—Å—ã–ª–∫–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Ä–∞–∑–ª–∏—á–Ω—ã–µ –Ω–æ–º–µ—Ä–∞ inode, —á–µ–º –æ—Å–Ω–æ–≤–Ω—ã–µ —Ñ–∞–π–ª—ã.
- Soft —Å—Å—ã–ª–∫–∏ —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è –ø–æ–ª–µ–∑–Ω—ã–º–∏, –µ—Å–ª–∏ –∏—Å—Ö–æ–¥–Ω—ã–π —Ñ–∞–π–ª –±—ã–ª —É–¥–∞–ª–µ–Ω.
- Soft —Å—Å—ã–ª–∫–∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Å–æ–∑–¥–∞–Ω—ã –∏–∑ –∫–∞—Ç–∞–ª–æ–≥–æ–≤.
- Soft —Å—Å—ã–ª–∫–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–æ–∑–¥–∞–Ω–∞ –Ω–∞ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–∏ —Ñ–∞–π–ª–æ–≤—ã—Ö —Å–∏—Å—Ç–µ–º.

- –§–∞–π–ª–æ–≤–∞—è —Å–∏—Å—Ç–µ–º–∞ –¥–æ–ª–∂–Ω–∞ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å —Ä–∞–±–æ—Ç—É —Å hard —Å—Å—ã–ª–∫–∞–º–∏.

### –ß—Ç–æ —Ç–∞–∫–æ–µ —Ñ–∞–π–ª–æ–≤—ã–π –¥–µ—Å–∫—Ä–∏–ø—Ç–æ—Ä, —á—Ç–æ —Ç–∞–º —Ö—Ä–∞–Ω–∏—Ç—Å—è?
